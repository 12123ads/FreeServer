# FreeServer
最近新出了一个叫 [阿贝云](http://www.abeiyun.com/free/) 的云服务器厂商,推出了一款号称永久免费的服务器,比较恶心的是,每过几天就要进行一次延期(在推荐的网站发表一篇关于阿贝云的推广软文,然后将文章截图提交),于是写了这个自动延期的项目
# 项目架构
JDK1.8 + SpringBoot + Mybatis + Mysql
